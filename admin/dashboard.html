Google Calendar

Sincroniza automáticamente las reservas con el calendario de Google. Al confirmar una reserva, se creará un evento automáticamente.

Sin configurar
Estado: —
Expira: —
Calendario: primary

Pasos para configurar Google Calendar

  1. Ir a Google Cloud Console
  2. Crear un proyecto nuevo (o seleccionar uno existente)
  3. Habilitar Google Calendar API: APIs & Services > Enable APIs and Services
  4. Crear credenciales OAuth 2.0: Credentials > Create Credentials > OAuth client ID
  5. Tipo: Web application
  6. Agregar URI de redirección autorizado: —
  7. Copiar Client ID y Client Secret
  8. Editar el archivo backend/config/google-calendar.php y pegar los valores
  9. Ir a OAuth consent screen y agregar tu email como usuario de prueba
  10. Volver aquí y hacer clic en "Conectar con Google"
